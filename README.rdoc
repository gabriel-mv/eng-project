== README

=== Nome do Projeto: Buddy Me!

Heroku Application Link: https://eng-project.herokuapp.com/

<b>Descrição:</b> Rede Social destinada a pessoas que desejam compartilhar experiências baseada em convites para espaços ou atividades de interesse mútuo. A idéia é seria os usuários cadastrarem atividades que antes fariam sozinhos para compartilharem essa experiência com outras pessoas com mesmo interesse.
